CTF備忘録

2022年にインターンシップでCTFを初体験し、就活後から個人でチャレンジしました。
2023年は個人参加でしたが、今後はチームで参加してみたいです。

1. SECCON Beginners 2023

開催日 : 2023年6月上旬
得点 : 223pt ( Welcome, Crypto, Reversing×2 )
順位 : 379 / 778 位
skill : Python, C

CTFに無知過ぎてどのような問題が出るのか、どのように勉強するのかから調べました。
セキュリティ専門書の内容は理解できず、とりあえず常設CTFを解いて小手先のテクニックをいくつか身につけました。

2. vsCTF 2023

開催日 : 2023年9月下旬
得点 : 300pt ( Web, Misc×2 )
順位 : 264 / 600 位
skill : Python, C

このCTFで、小手先のテクニックだけでは成績が絶対に頭打ちになると感じました。
そこで、まずはWeb系の知識を増やそうと考えました。 ( とりあえず、HTML, JS, PHP )

3. SpookyCTF 2023

開催日 : 2023年10月下旬
得点 : 150pt ( Bin, Misc×2 )
順位 : 312 / 372 位
skill : HTML, JS, PHP, Python, C

HTML, JS, PHPの基礎知識を学びましたが、Web分野では得点できず。
基礎力を挙げたつもりなのに成績は落ちて残念でしたが、チャレンジできる問題や正解まで惜しい問題は増えていました。
JSとPHPの学習は継続した上で、得点力を高めるためのテクニックも集めようと思いました。

4. NFLabs. Cybersecurity Challenge for Students 2023

開催日 : 2023年11月下旬
得点 : 1100pt ( Web, DFIR×4, OSINT×3 )
順位 : 17 / 50位
skill : HTML, JS, PHP, Python, C, SQL

DFIR ( Wireshark, ログ解析 ) やOSINTの粘り強さが必要となる問題はかなり解けました。
参加メンバーがハイレベルでビビっていましたが、上位半分に入れて嬉しかったです。
TOP10入りを目指すためには、Web, Malware分野の対策が必要だと感じました。

Conclusion2023

今年の成長ポイント
  • 扱えるプログラミング言語の増加
  • 扱えるツールの増加 ( IDA, Wireshark, etc )
  • GoogleやChatGPTを用いた検索力
来年の学習目標
  1. Webセキュリティ ( Hack The Boxで演習 )
  2. マルウェア解析 ( CTFの過去問演習 )
  3. 開発力 ( モバイル開発に挑戦 )
