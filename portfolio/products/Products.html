1. 介護支援Webアプリケーション

アプリの概要

介護分野における外国人労働者の困難を少しでも取り除くことを目指したアプリです。ローカル環境で作成済み。アプリの使用方法は、こちらをご覧ください！

アプリの機能と利点

  • 翻訳機能
    • 高齢者が介護者に求めていることの大筋を理解できる
    • 外国人労働者の主要な母国語に翻訳できる

  • ナースコール機能
    • 介護者は高齢者の要望・緊急性を見て対応順番を決められる
    • 高齢者が口頭では伝えにくい要望を介護者に伝えられる ( おむつ替えなど )

改善点

  • ナースコール機能の管理者ページの多言語対応
  • モバイルアプリの作成

3. 競馬AI Webアプリケーション

アプリの概要

ユーザーの好みに合わせた競馬の着順予想ができるアプリです。現在作成中。アプリの使用方法は、こちらをご覧ください！

解析方法

  • netkeiba.comから過去のレース情報をWebスクレイピング
  • 各レースにおいてオッズ, 馬番, 馬体重, 脚質が「着順」と関係するかを重回帰分析で確認
  • レースごとに着順予想計算式を算出し、データベースに格納
  • ユーザの重要視ポイントと計算式を組み合わせて、着順予想

改善点

  • レース情報の追加 ( 場所, 距離 )
  • 重要視ポイントの追加 ( 血統, ジョッキー, レーティング, etc )
  • 馬体チェック機能
  • Tensorflow, MoveNetを利用した動作解析機能
